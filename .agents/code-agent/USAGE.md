# CodeAgent 使用示例

## 🚀 快速开始

### 方式一：通过 sessions_spawn 调用

```javascript
// 在对话中调用
sessions_spawn(
  agentId="code-agent",
  task="设计并实现一个 RESTful API，支持用户注册、登录、信息修改功能",
  cleanup="keep",
  timeoutSeconds=3600
)
```

### 方式二：直接对话指定

```
@CodeAgent 请帮我设计一个电商网站的数据库 schema，需要包含：
- 用户表
- 商品表
- 订单表
- 支付记录表
请使用 MySQL，并给出建表语句和索引设计。
```

### 方式三：通过任务队列

```
任务类型：程序设计
优先级：高
描述：实现一个实时聊天功能，使用 WebSocket
交付物：设计文档 + 完整代码 + 测试用例
```

## 📋 典型场景

### 场景 1：新功能开发

**用户请求：**
```
需要为一个博客系统添加评论功能，支持：
- 用户发表评论
- 回复他人评论（嵌套）
- 点赞/点踩
- 评论审核
请设计数据库和 API。
```

**CodeAgent 工作流程：**
1. 需求分析 → 输出功能规格文档
2. 数据库设计 → 输出 ER 图和建表语句
3. API 设计 → 输出接口文档
4. 代码实现 → 调用 OpenCode 生成代码
5. 测试验证 → 编写并运行测试用例

**交付物：**
```
/blog-comments/
├── README.md           # 功能说明
├── database/
│   ├── schema.sql     # 建表语句
│   └── indexes.sql    # 索引设计
├── api/
│   ├── routes.js      # API 路由
│   ├── controllers.js # 控制器
│   └── validators.js  # 参数验证
├── tests/
│   └── comments.test.js # 测试用例
└── docs/
    └── API.md         # API 文档
```

### 场景 2：代码重构

**用户请求：**
```
这个文件有 500 行代码，太乱了。请帮我重构：
- 提取公共函数
- 减少嵌套层级
- 添加类型注释
- 优化性能
```

**CodeAgent 工作流程：**
1. 代码分析 → 识别问题和改进点
2. 重构方案 → 输出重构计划
3. 分步实施 → 逐步重构，每步验证
4. 测试确认 → 确保功能不变

**交付物：**
- 重构前后的代码对比
- 重构说明文档
- 测试报告

### 场景 3：技术选型

**用户请求：**
```
我们要开发一个实时数据监控面板，需要：
- 实时数据展示
- 图表可视化
- 告警通知
请推荐技术栈并说明理由。
```

**CodeAgent 输出：**
```markdown
## 技术选型建议

### 前端
- 框架：React + TypeScript
  理由：生态丰富，类型安全，组件化开发
- 图表：ECharts / Recharts
  理由：功能强大，支持实时更新
- 状态管理：Zustand
  理由：轻量，易用，适合中小型项目

### 后端
- 运行时：Node.js + Express
  理由：开发效率高，适合 I/O 密集型
- 实时通信：Socket.IO
  理由：成熟稳定，支持 fallback
- 数据库：PostgreSQL + Redis
  理由：PG 功能强大，Redis 适合缓存和实时数据

### 部署
- 容器：Docker
- 编排：Kubernetes（可选）
- 监控：Prometheus + Grafana
```

### 场景 4：Bug 排查

**用户请求：**
```
这个函数偶尔会返回错误结果，帮我找出问题：
[粘贴代码]
```

1. 代码审查 → 识别潜在问题
2. 日志分析 → 查看错误日志

## ⚙️ 配置选项

### 模型选择

```json
{
  "agent": "code-agent",
  "model": "qwen3.5-plus",  // 或 GLM-5、其他编程专用模型
  "thinking": "on",         // 开启深度思考
  "timeout": 3600           // 超时时间（秒）
}
```

### 工具权限

```json
{
  "tools": {
    "OpenCode": true,       // 代码生成
    "exec": true,           // 运行命令
    "browser": true,        // 查阅文档
    "read": true,           // 读取文件
    "write": true,          // 写入文件
    "edit": true            // 编辑文件
  }
}
```
